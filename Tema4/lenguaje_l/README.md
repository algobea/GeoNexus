# Analizador Léxico para Mini-Rust (Flex)

Este directorio contiene el código fuente de un analizador léxico construido con Flex para procesar un subconjunto básico del lenguaje Rust.

## Requisitos Previos
El entorno de desarrollo requiere la instalación de Flex y el compilador GCC. En distribuciones basadas en Debian/Ubuntu (como Linux Mint), ejecuta:
```bash
sudo apt update
sudo apt install flex gcc
